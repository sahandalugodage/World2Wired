# 🌐 World2Wired

> **Transforming semantic concepts into rapid wired prototypes.**

World2Wired is a cutting-edge design engine built for architects, developers, and product managers. It bridges the gap between abstract ideas and visual interfaces by mapping semantic logic directly to professional UI components.

---

## ✨ Key Features

- **🚀 Neural Architect**: Leveraging Llama-3.3-70b logic to translate textual descriptions into component maps.
- **🎨 Dynamic Theme Engine**: Real-time control over radius, glassmorphism depth, and accent palettes.
- **📊 Integrated Analytics**: Visualize component distribution and design integrity scores.
- **🛠️ Professional Component Library**: A robust set of pre-built, theme-aware components (Hero, Stats, Tables, Charts, and more).
- **📦 Multi-Format Export**: Export your designs as production-ready HTML/CSS or React code snippets.

## 🛠️ Technology Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 7](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Intelligence**: Groq Llama-3.3-70b Integration

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/sahandalugodage/World2Wired.git
   cd World2Wired
   ```

2. **Install dependencies**
   ```bash
   cd prototype
   npm install
   ```

3. **Setup Environment Variables**
   Create a `.env` file in the `prototype` directory:
   ```env
   VITE_GROQ_API_KEY=your_api_key_here
   ```

4. **Run Development Server**
   ```bash
   npm run dev
   ```
   Or use the included automation script:
   ```powershell
   ./launch_word2wired.bat
   ```
